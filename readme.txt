== Description ==

This plugin is used to easily access and insert videos hosted in the YouOnVideo platform into your WordPress posts.
YouOnVideo plugin is integrated with the WordPress Media Upload allowing an easy insertion of video.

You can filter your videos by channel and playlist. Videos are presented with thumbnail images for an easier selection.

The videos provided by YouOnVideo are responsive and adapt to the layout and size of your page.

Note that in order to use this plugin you must have a YouOnVideo account. Please visit [YouOnVideo](http://video.youongroup.com/ "YouOnVideo") to create an account.

== Installation ==

1. You can install YouOnVideo plugin via WordPress.org plugin directory, by uploading the zip file in 'Add Plugin' menu in WordPress or directly uploading the files to `/wp-content/plugins/` directory in your server.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to YouOnVideo menu and insert your API key and Account token provided by YouOnVideo.
4. Click in 'Update Options' and you're ready to go.

== Frequently Asked Questions ==

= Where can I get a YouOnVideo account? =

You must contact [YouOnVideo](http://video.youongroup.com/ "YouOnVideo") to create an account.

= I already have an account. Where can I get an API key and Account token? =

You must log in at [Backstage](https://stage.youongroup.com/video/home/ "Backstage") and access API tab under [Social Settings](https://stage.youongroup.com/settings/social/api/ "Social Settings").

== Screenshots ==

1. After complete the installation of the plugin (see Installation for more details) you must insert your API key and Account token (see Frequently Asked Questions for more details).

2. When creating/editing a post you can access your videos by clicking in 'Add Media' button and selecting the 'YouOnVideo' option. Select a Channel and a Playlist and the corresponding videos will appear. After selecting the video click in the 'Insert into post' button.

3. You must not change the code generated by the YouOnVideo plugin. This code will be transformed into the video when you Preview or Publish the post.

4. The final result. Your video is now published in your page.
